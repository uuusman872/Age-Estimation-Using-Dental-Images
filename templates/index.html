Let’s discuss about our Model

The data-set contains 786 x-ray images that were collected from GitHub containing 4 classes between age 20 to 60. The data-set is divided into Train and Test Sets for model training and testing. The data-set contains 743 training images and 25 test images.

Our Model help you to grow your Technology

The model is trained using resnet50 on these images. ResNet50 is a variant of ResNet model which has 48 Convolution layers along with 1 Max Pool and 1 Average Pool layer. It has 3.8×10^9 Floating points operations. It is a widely used Res Net model
